# Markdown 编辑器 - 运行环境要求与兼容性说明

## 📋 系统要求

### ✅ 支持的操作系统

#### Windows
- **最低版本**: Windows 7 SP1 (64位)
- **推荐版本**: Windows 10/11 (64位)
- **架构**: x86_64 (AMD64) 仅支持64位系统

#### ⚠️ 不支持的系统
- ❌ Windows XP / Vista
- ❌ Windows 32位系统
- ❌ macOS (需要重新编译)
- ❌ Linux (需要重新编译)

## 🔧 依赖组件要求

### Python 环境（开发版）
- **Python 版本**: ≥ 3.8 (推荐 3.10+)
- **必需库**:
  ```
  markdown==3.4.3
  PyQt6>=6.2.0
  ```

- **无需安装 Python**
- **自动包含所有依赖**
- **首次启动可能需要几秒加载时间**

## ⚠️ 已知兼容性风险

### 1. QtWebEngine 相关问题（高风险）

**问题描述**:
- QtWebEngine 基于 Chromium，对系统有较高要求
- 在某些旧系统上可能导致闪退或无法启动

**可能出问题的场景**:
- ❌ Windows 7 缺少必要更新（KB2533623等）
- ❌ 显卡驱动过旧或不支持 OpenGL 2.0+
- ❌ 系统缺少 Visual C++ Redistributable 2015-2022
- ❌ 防病毒软件拦截 QtWebEngineProcess.exe

**解决方案**:
```powershell
# 1. 安装最新 Windows 更新
# 2. 安装 Visual C++ Redistributable
https://aka.ms/vs/17/release/vc_redist.x64.exe

# 3. 更新显卡驱动
# 4. 将程序添加到杀毒软件白名单
```

---

### 2. 内存占用问题（中风险）

**问题描述**:
- QtWebEngine 是完整的浏览器引擎，内存占用较大
- 处理大型 Markdown 文件时可能占用 500MB+ 内存

**建议配置**:
- **最低内存**: 4GB RAM
- **推荐内存**: 8GB+ RAM
- **大文件处理**: 建议 < 10MB 的 Markdown 文件

---

### 3. 字体渲染问题（低风险）

**问题描述**:
- 不同系统字体渲染效果可能不同
- Windows 7 可能缺少某些新字体

**影响**:
- 预览区显示效果略有差异
- 不影响功能使用

---

### 4. PDF 导出依赖（中风险）

**问题描述**:
- PDF 导出依赖 PyMuPDF 和 Pillow
- 如果缺少这些库，导出功能会失败

**错误提示**:
```
缺少依赖: PyMuPDF 或 Pillow
```

**解决方案**:
- 打包版本已包含所有依赖
- 开发版本需手动安装: `pip install PyMuPDF Pillow`

## 🛡️ 稳定性增强措施

### 已实现的保护机制

1. **异常捕获**
   - 所有关键操作都有 try-except 保护
   - 错误信息记录到 `error.log`

2. **日志系统**
   - 详细记录运行状态
   - 便于排查问题

3. **资源清理**
   - 临时文件自动清理
   - 防止磁盘空间耗尽

4. ** graceful degradation**
   - 图标加载失败不影响启动
   - 配置文件损坏使用默认值

## 🔍 常见问题排查

### 问题1: 启动即闪退

**可能原因**:
1. 系统缺少 VC++ 运行时库
2. QtWebEngine 初始化失败
3. 权限不足

**解决方法**:
```powershell
# 1. 检查 error.log 文件
type error.log

# 2. 以管理员身份运行
# 右键 -> 以管理员身份运行

# 3. 安装 VC++ Redistributable
# 下载链接见上文
```

---

### 问题2: 预览区空白

**可能原因**:
1. QtWebEngine 插件加载失败
2. 路径包含中文或特殊字符
3. 防火墙阻止本地服务

**解决方法**:
```powershell
# 1. 确保路径不含特殊字符
# 2. 关闭防火墙测试
# 3. 查看 error.log 中的 WebEngine 相关错误
```

---

### 问题3: PDF 导出失败

**可能原因**:
1. PyMuPDF 未正确安装
2. 文件被其他程序占用
3. 磁盘空间不足

**解决方法**:
```powershell
# 1. 检查依赖
python -c "import fitz; import PIL"

# 2. 关闭占用文件的程序
# 3. 检查磁盘空间
```

---

### 问题4: 图片无法显示

**可能原因**:
1. 图片路径不正确
2. 图片格式不支持
3. 相对路径解析失败

**解决方法**:
```markdown
<!-- 使用正确的相对路径 -->
![图片](./images/test.png)

<!-- 或使用绝对路径 -->
![图片](D:/path/to/image.png)
```
