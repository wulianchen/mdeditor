## 📦 核心框架

### 1\. PyQt6 \(≥6\.2\.0\)

- **许可证**: GPL v3 / 商业许可

- **用途**: Python GUI 框架，提供窗口、控件等界面组件

- **官网**: https://www\.riverbankcomputing\.com/software/pyqt/

- **引入方式**: \`from PyQt6\.QtWidgets import \.\.\.\`

### 2\. PyQt6\-WebEngine \(≥6\.2\.0\)

- **许可证**: GPL v3 / 商业许可

- **用途**: 基于 Chromium 的 Web 渲染引擎，用于 Markdown 预览

- **官网**: https://www\.riverbankcomputing\.com/software/pyqtwebengine/

- **引入方式**: \`from PyQt6\.QtWebEngineWidgets import QWebEngineView\`

## 📝 Markdown 处理

### 3\. markdown \(3\.4\.3\)

- **许可证**: BSD 3\-Clause

- **用途**: Markdown 文本解析和 HTML 转换

- **官网**: https://python\-markdown\.github\.io/

- **引入方式**: \`import markdown\`

### 4\. Pygments

- **许可证**: BSD 2\-Clause

- **用途**: 代码语法高亮引擎（通过 markdown 扩展自动安装）

- **官网**: https://pygments\.org/

- **引入方式**: \`from markdown\.extensions\.codehilite import CodeHiliteExtension\`

---

## 🖼️ PDF 和图片处理

### 5\. PyMuPDF \(fitz\)

- **许可证**: AGPL v3 / 商业许可

- **用途**: PDF 文件处理和图片导出

- **官网**: https://pymupdf\.readthedocs\.io/

- **引入方式**: \`import fitz\`

### 6\. Pillow \(PIL\)

- **许可证**: HPND \(Historical Permission Notice and Disclaimer\)

- **用途**: 图像处理库，用于长图导出和裁剪

- **官网**: https://pillow\.readthedocs\.io/

- **引入方式**: \`from PIL import Image\`

### 7\. NumPy

- **许可证**: BSD 3\-Clause

- **用途**: 数值计算库，支持图像处理（通过 Pillow/PyMuPDF 间接依赖）

- **官网**: https://numpy\.org/

- **引入方式**: 间接依赖
