# 快乐马Markdown编辑器 \- 产品简介

## 一、工具定位

快乐马 Markdown 编辑器是专为 AI 时代打造的高效文档处理工具，专注解决 AI 生成 Markdown 内容的落地排版难题。可对 AI 产出的纯文本 Markdown 内容一键美化、自定义排版样式，快速生成层级清晰、格式规整的专业文档，支持 PDF、长图、图集多格式导出，完美适配电脑、手机、平板全终端浏览排版需求。

## 二、核心优势

### 1\. 高颜值样式体系

内置 20\+ 专业主题风格，涵盖企业级 UI、程序员专属、阅读优化等类别，其中企业级主题包含 Ant Design、Material Design 等规范样式，程序员专属主题聚焦极客风与暗黑风，可根据需求一键切换，轻松解决 AI 生成内容视觉层次不足、样式简陋的问题，同时支持代码高亮与个性化样式调整。

### 2\. 全场景导出能力

支持 PDF、长图、图集三种导出模式，适配不同使用场景：PDF 导出适配正式归档、打印需求，长图导出适配社交媒体分享，图集导出适配 PPT 插入与教学场景，所有导出模式均遵循 “所见即所得” 原则，兼顾实用性与专业性。

- **PDF 导出**：A4 比例智能分页，高分辨率，适配正式归档、客户交付等场景；

- **长图导出**：无缝拼接，高清无锯齿，适配微信公众号、小红书等社交分享场景；

- **图集导出**：按页拆分、规范命名，可直接插入 PPT、制作教学课件。

### 3\. 专业级编辑体验

- 支持实时双栏预览、滚动同步，完整兼容 Markdown 语法，代码块自动识别 60\+ 编程语言并高亮，表格与脚注展示规范，满足专业排版需求；

- 代码块自动识别 60\+ 编程语言并高亮，表格渲染、脚注展示等功能全覆盖；

- 可自定义文档宽度（20%\-100%）、基准字号（12px\-24px）、字体及字重，满足个性化排版需求。

### 4\. 安全高效的使用体验

全程本地处理，无需上传云端，保障隐私安全，支持离线使用，搭配快捷键操作与配置自动保存功能，大幅提升编辑与导出效率。

## 三、适配人群与场景
