# 快乐马 MarkDown 编辑器技术白皮书

## 一、产品定位

快乐马 MarkDown 编辑器是一款专为 AI 时代设计的轻量化、高性能文档处理工具，核心解决 MarkDown 文档编辑、美化及多格式导出需求，适配各类办公、创作场景，依托成熟技术栈保障运行稳定，兼顾专业性与实用性，无需复杂配置，开箱即用。

## 二、核心技术架构

### 1\. 核心技术栈选型

|类别|技术选型|说明|
|---|---|---|
|GUI 框架|PyQt6|支撑桌面端可视化界面，保障操作流畅性|
|Web 渲染|Qt WebEngine（基于 Chromium 内核）|实现 MarkDown 实时预览、HTML 渲染，还原度高|
|MarkDown 解析|Python\-Markdown|精准解析 MarkDown 语法，支持复杂格式排版|
|代码高亮|Pygments|支持多语言语法高亮，提升代码可读性|
|PDF 处理|PyMuPDF|高质量生成 PDF，支持自定义页面设置|
|图像处理|Pillow|实现长图拼接、尺寸调整，适配导出需求|

### 2\. 核心技术实现细节

#### 2\.1 双栏实时预览架构

采用“编辑区\+预览区”双栏布局，编辑区（QTextEdit）负责输入与修改，预览区（QWebEngineView）负责实时渲染，通过信号槽机制实现两者滚动同步，确保编辑内容与预览效果一致，无延迟、无卡顿。

关键实现代码（精简版）：

```python
def sync_scroll(self):
    # 同步编辑区与预览区滚动位置
    editor_scroll = self.editor.verticalScrollBar().value()
    preview_scroll = self.preview.verticalScrollBar().value()
    # 计算滚动比例，保持同步
    ratio = self.editor.width() / self.preview.width()
    self.preview.verticalScrollBar().setValue(int(editor_scroll * ratio))
```

#### 2\.2 多格式导出实现

支持 PDF、长图、图集三种导出模式，适配不同使用场景，核心代码（精简版）：

```python
def export_pdf(self, output_path):
    # PDF 导出（正式归档）
    printer = QPrinter()
    printer.setOutputFormat(QPrinter.PdfFormat)
    printer.setOutputFileName(output_path)
    # 关联预览区内容，确保格式一致
    self.renderer.render(self.preview, printer)

def export_long_image(self, output_path):
    # 长图导出（社交分享）
    pixmap = QPixmap()
    self.renderer.render(pixmap)
    pixmap.save(output_path, "PNG")

def export_gallery(self, output_path):
    # 图集导出（PPT 插入、逐页展示）
    for page in self.pages:
        pix = QPixmap()
        self.renderer.render(page, pix)
        pix.save(f"{output_path}/page_{page.num}.png")
```

#### 2\.3 配置持久化

通过 JSON 格式保存用户配置（主题、窗口位置、字体设置），下次启动自动加载，无需重复配置，核心代码：

```python
def save_config(self):
    config = {
        "theme": self.current_theme,
        "window_geometry": self.save_geometry(),
        "font": self.font().family()
    }
    with open("config.json", "w", encoding="utf-8") as f:
        json.dump(config, f, ensure_ascii=False, indent=2)
```

## 三、技术优势与注意事项

### 3\.1 核心优势

- **轻量化高效**：基于 PyQt6 构建，体积小、启动快，无冗余依赖，适合桌面端快速部署

- **渲染精准**：Chromium 内核保障 HTML 与 MarkDown 语法完美适配，无格式错乱

- **多场景适配**：支持正式归档（PDF）、社交分享（长图）、课件插入（图集），覆盖全使用场景

- **易操作**：拖拽打开、一键导出，无需复杂配置，新手可快速上手

### 3\.2 注意事项

- 解压后直接启动可执行文件，无需额外安装环境，绿色免配置

- 确保 MarkDown 文件编码为 UTF\-8，避免中文乱码

- 导出时确保预览区内容完整，避免未渲染完成即导出导致格式错乱

- 窗口大小可拖动调整，避免挤压控件导致功能异常

## 四、总结

快乐马 MarkDown 编辑器以 PyQt6 为核心框架，依托 Chromium 渲染技术，实现 MarkDown 文档的高效编辑、精准预览与多格式导出，兼顾专业性与易用性。相较于传统编辑器，其轻量化部署、零复杂配置的特点，更适配 AI 时代快速创作需求，同时通过配置持久化、多场景导出，大幅提升文档处理效率，适用于办公、教学、技术分享等各类场景，实现“打开即编辑、导出即可用”的核心需求。
